The Game of Life

This is a HTML 5 Canvas and JavaScript implementation of John Conway's Game of Life. It makes use of the JavaScript libraries jQuery and RequireJS.

Source code can be found online at GitHub and more information on my blog.

You can pick from starting patterns below or edit the board by clicking it's cells.

Your browser does not support running Game of Life. Please upgrade to a more recent version that supports HTML 5 canvas.
